조건문(condition)

    프로그래밍 코드가 해석되는 순서를 제어하는 제어문의 한 종류
    작성된 조건식에 따라서 수행되는 코드를 변경하는 구문

if문

    
        if(조건식){
            조건식이 true인 경우 실행할 코드;
        }
    

    * 조건식 : 결과가 true, false인 식(연산)
    * 조건문 : 조건식의 결과에 따라 실행을 제어하는 구문(if, switch)

입력받은 수가 양수인지 아닌지 검사

if - else 문

        
            if(조건식) {
                조건식 true면 수행 코드;
            }

            else {
                if 조건식 false면 수행 코드;
            }
        

        * else 구문에는 조건식을 작성하는 ()가 없다!

0 ~ 100 사이 난수 발생시켜 홀짝 판별하기

if - else if - else

        
            if(조건식) {
                조건식 true면 실행;
            } else if(조건식) {
                앞선 if문 false이고 elif 조건식 true인 경우;
            } else {
                다 false이면 실행;
            }
        

        * else if 횟수 제한X 원하는 만큼 작성O
        * 앞선 if/else if 조건 false인 경우 이후 else if에서 해당 범위 제외됨.

-3 ~ 3 사이 난수 발생시켜 양수, 음수, 0 판별

어린이, 청소년, 성인 구분하기

    나이 입력 받아 
    0 ~ 13 : 어린이
    14 ~ 19 : 청소년
    20 ~ 150 : 성인
    0 미만 150 초과 : 잘못 입력 했어
    미입력 : 값 입력해

switch, case, default

    switch문
    - 식 하나의 결과로 많은 경우의 수를 처리할 때 사용하는 조건문
      -> 식의 결과에 맞는 case 구문이 수행됨

    
        switch(식){ // 식의 결과는 정수 또는 문자열
     
        case 결과1 : 결과1의 수행 코드; break;
        case 결과2 : 결과2의 수행 코드; break;
        case 결과3 : 결과3의 수행 코드; break;
        ...
          
        default : 모든 case가 만족하지 못할 때 수행할 코드
        }
