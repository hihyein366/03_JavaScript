비교 연산자

    == , !=, >, <, >=, <=

    - 좌변과 우변을 같다, 같지않다, 초과, 미만, 이상, 이하 비교해서 
      맞으면 true, 아니면 false 반환하는 연산자
      (연산 결과로 boolean(논리 값) 반환)

    - 복합 기호 사용할 때 등호(=)는 무조건 오른쪽에 작성(>=, <=)

입력된 두 값이 같은지 확인



좌변에 입력한 수가 우변에 입력한 수보다 큰지 확인



배수가 맞는지 확인




5는 2의 배수 : false

복합 대입 연산자(+=, -=, *=, /=, %=)

    - 기존 변수에 저장 되어있는 값에 새롭게 대입되는 값의 산술 연산 결과를 대입

      (증감 연산자는 1씩 증가, 감소, 
       복합 대입 연산자는 원하는 만큼 증가, 감소)
      ex)
      let temp = 10;

      temp += 5; // temp = temp + 5; (누적/차감 효과)

현재 값 : 0

논리 연산자 ( &&, ||, ! )

    AND 연산자 (A && B)
    - A, B 모두 true일 때 true
    - 문장에서 그리고(이고), 이면서, 부터, 까지, 사이 있으면 AND

    OR 연산자 (A || B)
    - A 또는 B, 둘 다 true일 때 true
    - 문장에서 또는,  이거나(거나) 있으면 OR

    논리 부정(NOT) 연산자(!)
    - 논리 값을 반대로 바꾸는 연산자
    
    ex) let test = true;
        !test; // !true = false;

삼항 연산자 (조건식 ? true : false)

    - 조건식의 결과에 따라서 수행되는 코드가 다른 연산자
      (조건식은 항상 결과로 true 또는 false가 반환되도록 작성)
